# NoeliaYoga V2

Segunda versión del sitio web de NoeliaYoga, rediseñado con **React + Vite + Tailwind CSS**.

## Stack

- React + Vite
- Tailwind CSS
- Logo e imágenes personalizados

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notas

- Rediseño completo con Tailwind CSS
- Incluye logos y assets gráficos
- Ver `V1` para la versión original
